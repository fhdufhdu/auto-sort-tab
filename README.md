# 개요
혹시 난잡한 탭 때문에 고민이 많으셨나요?

원하는 탭 찾기가 불편하셨나요?

도메인 별로 자동정렬된 탭으로 이전보다 쉽게 찾아보는 건 어떨까요?

# 사용 방법
1. git clone을 한다. (혹은 zip 다운 후 압축 해제)
2. 크롬 확장 프로그램 페이지로 이동 -> 압축해제된 확장 프로그램 로드 선택
3. clone이나 압축 해제한 디렉토리 선택
4. 적용 완료 - 새로 탭을 열때마다 자동으로 도메인이 정렬됨
